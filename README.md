# Sample-Project-Git-Workflow

Step 1:
    - Ensure you are in the correct folder
    - Set up upstream:
    - (ONLY ONCE) git remote add upstream
    ```<url of upstream>```

Step 2:
    - (EVERYDAY) Make sure you are up to date.
    - Command: git pull upstream main

Step 3:
    - Create a new branch
    -Command: git checkout -b 
    ```<your branch name>```

Step 4:
    - Work on the feature
    - Make changes
    - Code
